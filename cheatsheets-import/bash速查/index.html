Bash 实用快捷命令 实用

  • alias ll='ls -la' 创建别名
  • unalias ll 删除别名
  • alias 列出所有别名
  • history 显示历史命令
  • history -c 清空历史
  • history | grep pattern 搜索历史
  • source ~/.bashrc 重新加载配置
  • . ~/.bashrc 重新加载配置(简写)
  • export VAR=value 导出环境变量
  • env 显示所有环境变量
  • echo $VAR 显示变量值
  • which command 查找命令位置>Ctrl + A 跳转到行首 快速定位到命令开始
  • Ctrl + E 跳转到行尾 快速定位到命令结尾
  • Ctrl + U 删除光标前内容 清空当前输入
  • Ctrl + K 删除光标后内容 删除到行尾
  • Ctrl + W 删除光标前一个单词 按单词删除
  • Alt + D 删除光标后一个单词 向前删除单词
  • Alt + B 向后移动一个单词 按单词后退
  • Alt + F 向前移动一个单词 按单词前进
  • Ctrl + B / ← 向后移动一个字符
  • Ctrl + F / → 向前移动一个字符
  • Ctrl + D 删除光标处字符 相当于Delete键
  • Ctrl + H 删除光标前字符 相当于Backspace
  • Alt + C 大写当前单词 单词首字母大写
  • Alt + U 大写光标后单词 整个单词大写
  • Alt + L 小写光标后单词 整个单词小写

Bash 命令行编辑 基础

  • /pattern 向下查找模式
  • ?pattern 向上查找模式
  • n 下一个匹配
  • N 上一个匹配
  • * 查找当前词的下一个
  • # 查找当前词的上一个
  • g* 查找包含当前词的下一个
  • g# 查找包含当前词的上一个
  • gd 跳到局部变量定义
  • gD 跳到全局变量定义
  • :%s/old/new/g 全局替换
  • :%s/old/new/gc 带确认的全局替换
  • :%s/old/new/gi 忽略大小写替换
  • :s/old/new/g 当前行替换
  • :10,20s/old/new/g 指定行范围替换
  • :'<,'>s/old/new/g 可视区域替换
  • :g/pattern/d 删除匹配行
  • :v/pattern/d 删除不匹配行
  • :g/pattern/s/old/new/g 匹配行内替换

Bash 环境变量 环境

  • $PATH 可执行文件搜索路径
  • $HOME / ~ 用户主目录
  • $PWD 当前工作目录
  • $OLDPWD 上一个工作目录
  • $USER 当前用户名
  • $SHELL 当前shell
  • $PS1 主提示符
  • $PS2 续行提示符
  • $HISTSIZE 历史命令缓存数量
  • $HISTFILE 历史文件路径
  • $HISTCONTROL 历史控制选项
  • $EDITOR 默认编辑器
  • $TERM 终端类型
  • $LANG 系统语言
  • $ 当前进程ID
  • $? 上一命令退出码
  • $! 后台进程ID
  • $0 脚本名称
  • $1, $2, ... 脚本参数
  • $# 参数个数
  • $@ 所有参数
  • $* 所有参数(单字符串)

Bash 通配符与重定向 通配符

  • * 匹配零个或多个字符
  • ? 匹配单个字符
  • [abc] 匹配字符集中任一字符
  • [a-z] 匹配范围内字符
  • [^abc] 匹配除指定字符外的字符
  • {a,b,c} 花括号展开
  • {1..10} 序列展开
  • ~ 用户主目录
  • ~user 指定用户主目录
  • > 重定向输出
  • >> 追加输出
  • < 重定向输入
  • << Here文档
  • <<< Here字符串
  • | 管道连接命令
  • & 后台运行
  • && 逻辑AND
  • || 逻辑OR
  • ; 命令分隔符
  • 2>&1 错误重定向到标准输出
  • &> 重定向所有输出
  • /dev/null 空设备

Bash 历史导航 历史

  • Ctrl + R 反向搜索历史命令 增量搜索
  • Ctrl + S 正向搜索历史命令 需要stty -ixon
  • Ctrl + G 退出搜索模式 取消历史搜索
  • ↑ / Ctrl + P 上一条历史
  • ↓ / Ctrl + N 下一条历史
  • !! 重复执行上一次命令
  • !n 执行第n条历史记录 n为行号
  • !-n 执行倒数第n条记录
  • !string 执行最近以string开头的命令
  • !?string 执行最近包含string的命令
  • ^old^new 替换上一命令中的old为new
  • !!:p 打印但不执行上一命令
  • !$ 上一命令的最后参数
  • !^ 上一命令的第一个参数
  • !* 上一命令的所有参数
  • Alt + . / Alt + _ 插入上一命令最后参数
