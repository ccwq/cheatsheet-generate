窗格管理 (Panes)

分屏操作

  • 纵向分屏：Alt + Shift + =
  • 横向分屏：Alt + Shift + -
  • 关闭当前窗格：Ctrl + Shift + W

焦点切换

  • 在窗格间切换：Alt + ← ↑ → ↓

调整大小

  • 调整窗格大小：Alt + Shift + ← ↑ → ↓
基于 Windows Terminal 官方文档“Windows 终端中的窗格”整理。

标签页管理 (Tabs)

标签页操作

  • 新建标签页：Ctrl + Shift + T
  • 关闭标签页：Ctrl + Shift + W
  • 下一个标签页：Ctrl + Tab
  • 上一个标签页：Ctrl + Shift + Tab
标签页是多会话并行工作的核心，高效切换提升生产力。

剪贴板与文本 (Clipboard & Text)

复制与粘贴

  • 复制：Ctrl + Shift + C
  • 粘贴：Ctrl + Shift + V 或 Shift + Insert

查找与翻页

  • 查找：Ctrl + Shift + F
  • 上翻页：Ctrl + Shift + PageUp
  • 下翻页：Ctrl + Shift + PageDown

外观与主题 (Appearance)

主题与配色方案

  • 在 settings.json 中编辑 "schemes" 列表

背景与毛玻璃

  • 设置背景图："backgroundImage": "路径"
  • 启用毛玻璃："useAcrylic": true
  • 毛玻璃透明度："acrylicOpacity": 0.75

配置文件 (settings.json)

常用字段

{
  "defaultProfile": "{GUID}",
  "profiles": {
    "list": [
      {
        "guid": "{GUID}",
        "name": "PowerShell",
        "commandline": "pwsh.exe",
        "startingDirectory": "%USERPROFILE%"
      }
    ]
  },
  "actions": [
    {
      "command": "splitPane",
      "keys": "alt+shift+="
    }
  ]
}
示例展示分屏快捷键绑定和默认配置。

自定义操作 (Actions)

常用命令

  • splitPane：分割窗格
  • closePane：关闭窗格
  • resizePane：调整窗格大小
  • newTab：新建标签

键绑定示例

{
  "action": "resizePane",
  "keys": "alt+shift+right"
}

窗口与全屏 (Window & Focus)

全屏切换

  • 全屏/退出：F11 或 Alt + Enter

放大/缩小字体

  • 放大：Ctrl + 鼠标滚轮 ↑
  • 缩小：Ctrl + 鼠标滚轮 ↓

焦点模式

  • 焦点模式（隐藏标签页等）：Alt + F11

故障排除 (Troubleshooting)

重置设置

  • 删除或重命名 %LOCALAPPDATA%\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState\settings.json 重置为默认

日志查看

  • 运行 wt --verbose 查看启动日志
