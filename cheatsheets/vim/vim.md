# Vim 速查

## 📝 进入和退出编辑模式
---
lang: bash
emoji: 📝
link: https://www.vim.org/docs.php
desc: 模式切换是 Vim 的核心概念。
---

### 进入插入模式

- `i` : 在光标前插入
- `a` : 在光标后插入
- `I` : 在行首插入
- `A` : 在行尾插入
- `o` : 在下方新建一行
- `O` : 在上方新建一行
- `s` : 删除光标字符并插入
- `S` : 删除整行并插入

### 退出插入模式

- `Esc` : 退回到普通模式
- `Ctrl+[` : 退回到普通模式（等价于 Esc）

### 退出保存

- `:wq` : 保存并退出
- `ZZ` : 保存并退出
- `:q!` : 强制退出（不保存）
- `ZQ` : 不保存退出

## 🖱️ 移动光标
---
lang: bash
emoji: 🖱️
link: https://www.vim.org/docs.php
desc: Vim 的高效移动方式。
---

### 字符移动

- `h` : 左移
- `j` : 下移
- `k` : 上移
- `l` : 右移

### 词级移动

- `w` : 移到下一个词首
- `b` : 移到上一个词首
- `e` : 移到词尾
- `ge` : 移到前一个词尾

### 行内移动

- `0` : 移到行首
- `^` : 移到行首（非空白）
- `$` : 移到行尾
- `g_` : 移到行尾（非空白）

### 文件内移动

- `gg` : 移到文件开头
- `G` : 移到文件结尾
- `:n` : 跳转到第 n 行
- `:nG` : 跳转到第 n 行

### 屏幕移动

- `Ctrl+f` : 向下翻页
- `Ctrl+b` : 向上翻页
- `Ctrl+d` : 向下半页
- `Ctrl+u` : 向上半页
- `H` : 移到屏幕顶部
- `M` : 移到屏幕中间
- `L` : 移到屏幕底部

### 括号匹配

- `%` : 跳转到匹配的括号

- `R` : 进入替换模式
- `cw` : 修改词
- `cc` : 修改整行
- `C` : 修改到行尾

### 撤销

- `u` : 撤销上一次操作
- `Ctrl+r` : 重做
- `.` : 重复上一次操作

## 🔍 搜索和替换
---
lang: bash
emoji: 🔍
link: https://www.vim.org/docs.php
desc: 查找和批量替换。
---

### 搜索

- `/pattern` : 向下搜索
- `?pattern` : 向上搜索
- `n` : 下一个匹配
- `N` : 上一个匹配
- `*` : 搜索光标所在词（向下）
- `#` : 搜索光标所在词（向上）

### 替换

- `:s/old/new/` : 替换当前行第一个
- `:s/old/new/g` : 替换当前行全部
- `:%s/old/new/g` : 替换文件全部
- `:%s/old/new/gc` : 替换文件全部（确认）

### 正则模式

- `/\cpattern` : 忽略大小写
- `/\Cpattern` : 区分大小写
- `/\vpattern` : 很"魔法"正则
- `/\Vpattern` : 很"_literal"字面

---
lang: bash
emoji: 🔄
link: https://www.vim.org/docs.php
desc: 录制命令序列和位置标记。
---

### 宏录制

- `q a` : 开始录制宏到寄存器 a
- `q` : 结束录制
- `@ a` : 执行寄存器 a 的宏
- `@@` : 重复上次宏执行

### 标记

- `m a` : 在光标位置设置标记 a
- `` ` a`` : 跳转到标记 a
- `' a` : 跳转到标记 a 所在行

### 特殊标记

- ``.` : 上次修改位置
- `''` : 上次跳转前的位置

## 🪟 窗口和标签页
---
lang: bash
emoji: 🪟
link: https://www.vim.org/docs.php
desc: 分屏和标签页管理。
---

### 分屏

- `:sp` : 水平分屏
- `:vsp` : 垂直分屏
- `:sp filename` : 水平分屏打开文件
- `:vsp filename` : 垂直分屏打开文件

### 窗口操作（Ctrl+w 前缀）

- `Ctrl+w s` : 水平分割当前窗口
- `Ctrl+w v` : 垂直分割当前窗口
- `Ctrl+w h/j/k/l` : 切换到 左/下/上/右 窗口
- `Ctrl+w w` : 切换到下一个窗口
- `Ctrl+w q` : 关闭当前窗口
- `Ctrl+w =` : 所有窗口等宽高
- `Ctrl+w _` : 最大化当前窗口高度
- `Ctrl+w |` : 最大化当前窗口宽度

### 标签页

- `:tabnew` : 新建标签页
- `:tabe filename` : 在新标签页打开文件
- `gt` : 切换到下一个标签
- `gT` : 切换到上一个标签
- `:tabclose` : 关闭当前标签

## ⚙️ 配置选项
---
lang: bash
emoji: ⚙️
link: https://www.vim.org/docs.php
desc: 常用配置选项。
---

### 常用设置

- `:set nu` : 显示行号
- `:set nonu` : 隐藏行号
- `:set ai` : 自动缩进
- `:set noai` : 取消自动缩进
- `:set ic` : 搜索忽略大小写
- `:set noic` : 搜索区分大小写
- `:set ts=4` : Tab 宽度为 4
- `:set sw=4` : 缩进宽度为 4
- `:set et` : 用空格代替 Tab
- `:set wrap` : 自动换行
- `:set nowrap` : 不自动换行
- `:set hlsearch` : 高亮搜索结果
- `:set nuw=6` : 行号宽度

### 查看帮助

- `:help option-name` : 查看选项帮助

## 💡 实用技巧
---
lang: bash
emoji: 💡
link: https://www.vim.org/docs.php
desc: 提升效率的技巧。
---

### 大小写转换

- `gUU` : 整行转大写
- `guu` : 整行转小写
- `gUiw` : 光标所在词转大写
- `guiw` : 光标所在词转小写
- `~` : 切换光标处字符大小写

### 数字递增/递减

- `Ctrl+a` : 光标处数字加 1
- `Ctrl+x` : 光标处数字减 1

### 格式化

- `=` : 格式化当前代码块
- `gg=G` : 格式化整个文件

### 移动

- `gm` : 移到行中间
- `g;` : 跳转到上次修改位置
- `g,` : 跳转到下次修改位置
