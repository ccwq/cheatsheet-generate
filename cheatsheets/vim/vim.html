📝 进入和退出编辑模式 >>>

模式切换是 Vim 的核心概念。

进入插入模式

  • i：在光标前插入
  • a：在光标后插入
  • I：在行首插入
  • A：在行尾插入
  • o：在下方新建一行
  • O：在上方新建一行
  • s：删除光标字符并插入
  • S：删除整行并插入

退出插入模式

  • Esc：退回到普通模式
  • Ctrl+[：退回到普通模式（等价于 Esc）

退出保存

  • :wq：保存并退出
  • ZZ：保存并退出
  • :q!：强制退出（不保存）
  • ZQ：不保存退出

🖱️ 移动光标 >>>

Vim 的高效移动方式。

字符移动

  • h：左移
  • j：下移
  • k：上移
  • l：右移

词级移动

  • w：移到下一个词首
  • b：移到上一个词首
  • e：移到词尾
  • ge：移到前一个词尾

行内移动

  • 0：移到行首
  • ^：移到行首（非空白）
  • $：移到行尾
  • g_：移到行尾（非空白）

文件内移动

  • gg：移到文件开头
  • G：移到文件结尾
  • :n：跳转到第 n 行
  • :nG：跳转到第 n 行

屏幕移动

  • Ctrl+f：向下翻页
  • Ctrl+b：向上翻页
  • Ctrl+d：向下半页
  • Ctrl+u：向上半页
  • H：移到屏幕顶部
  • M：移到屏幕中间
  • L：移到屏幕底部

括号匹配

  • %：跳转到匹配的括号

✂️ 删除和修改文本 >>>

删除、替换、修改操作。

删除

  • x：删除光标处字符
  • X：删除光标前字符
  • dw：删除到词尾
  • dd：删除整行
  • D：删除到行尾
  • d$：删除到行尾
  • d0：删除到行首

替换

  • r：替换光标处字符（单个）
  • R：进入替换模式
  • cw：修改词
  • cc：修改整行
  • C：修改到行尾

撤销

  • u：撤销上一次操作
  • Ctrl+r：重做
  • .：重复上一次操作

📋 复制和粘贴 >>>

复制、剪切、粘贴。

复制

  • yw：复制到词尾
  • yy：复制整行
  • y$：复制到行尾
  • y0：复制到行首

粘贴

  • p：在光标后粘贴
  • P：在光标前粘贴

寄存器操作

  • "ayy：复制到寄存器 a
  • "ap：从寄存器 a 粘贴

🔍 搜索和替换 >>>

查找和批量替换。

搜索

  • /pattern：向下搜索
  • ?pattern：向上搜索
  • n：下一个匹配
  • N：上一个匹配
  • *：搜索光标所在词（向下）
  • #：搜索光标所在词（向上）

替换

  • :s/old/new/：替换当前行第一个
  • :s/old/new/g：替换当前行全部
  • :%s/old/new/g：替换文件全部
  • :%s/old/new/gc：替换文件全部（确认）

正则模式

  • /\cpattern：忽略大小写
  • /\Cpattern：区分大小写
  • /\vpattern：很"魔法"正则
  • /\Vpattern：很"_literal"字面

🔧 文本对象 >>>

快速选中词、句、段落、引号内等。

内部对象

  • iw：整个词（不含空格）
  • aw：整个词（含空格）
  • is：整个句子
  • ip：整个段落

引号内

  • i"：双引号内
  • i'：单引号内
  • `i`` : 反引号内

括号内

  • ib：圆括号内
  • i)：圆括号内
  • i]：方括号内
  • i}：大括号内
  • i>：尖括号内

配合动作

  • diw：删除光标所在词
  • ciw：修改光标所在词
  • yiw：复制光标所在词
  • da"：删除包括引号
  • ca"：修改包括引号

🔄 宏和标记 >>>

录制命令序列和位置标记。

宏录制

  • q a：开始录制宏到寄存器 a
  • q：结束录制
  • @ a：执行寄存器 a 的宏
  • @@：重复上次宏执行

标记

  • m a：在光标位置设置标记 a
  • ` a：跳转到标记 a
  • ' a：跳转到标记 a 所在行

特殊标记

  • ``.` : 上次修改位置
  • ''：上次跳转前的位置

🪟 窗口和标签页 >>>

分屏和标签页管理。

分屏

  • :sp：水平分屏
  • :vsp：垂直分屏
  • :sp filename：水平分屏打开文件
  • :vsp filename：垂直分屏打开文件

窗口操作（Ctrl+w 前缀）

  • Ctrl+w s：水平分割当前窗口
  • Ctrl+w v：垂直分割当前窗口
  • Ctrl+w h/j/k/l：切换到 左/下/上/右 窗口
  • Ctrl+w w：切换到下一个窗口
  • Ctrl+w q：关闭当前窗口
  • Ctrl+w =：所有窗口等宽高
  • Ctrl+w _：最大化当前窗口高度
  • Ctrl+w |：最大化当前窗口宽度

标签页

  • :tabnew：新建标签页
  • :tabe filename：在新标签页打开文件
  • gt：切换到下一个标签
  • gT：切换到上一个标签
  • :tabclose：关闭当前标签

⚙️ 配置选项 >>>

常用配置选项。

常用设置

  • :set nu：显示行号
  • :set nonu：隐藏行号
  • :set ai：自动缩进
  • :set noai：取消自动缩进
  • :set ic：搜索忽略大小写
  • :set noic：搜索区分大小写
  • :set ts=4：Tab 宽度为 4
  • :set sw=4：缩进宽度为 4
  • :set et：用空格代替 Tab
  • :set wrap：自动换行
  • :set nowrap：不自动换行
  • :set hlsearch：高亮搜索结果
  • :set nuw=6：行号宽度

查看帮助

  • :help option-name：查看选项帮助

🔀 常用命令组合 >>>

实用的高效操作组合。
  • cc：删除当前行并进入插入模式
  • A;：在行尾添加分号
  • ciw：删除当前单词并修改
  • yyp：复制当前行并粘贴到下方
  • V y：选中当前行
  • C：删除光标到行尾并进入插入模式
  • r x：替换当前光标字符
  • .：重复上次操作

💡 实用技巧 >>>

提升效率的技巧。

大小写转换

  • gUU：整行转大写
  • guu：整行转小写
  • gUiw：光标所在词转大写
  • guiw：光标所在词转小写
  • ~：切换光标处字符大小写

数字递增/递减

  • Ctrl+a：光标处数字加 1
  • Ctrl+x：光标处数字减 1

格式化

  • =：格式化当前代码块
  • gg=G：格式化整个文件

移动

  • gm：移到行中间
  • g;：跳转到上次修改位置
  • g,：跳转到下次修改位置
