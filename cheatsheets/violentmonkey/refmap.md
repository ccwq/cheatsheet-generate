# Violentmonkey 参考资源映射

## 官方文档

### 入门指南
- [Creating a userscript](https://violentmonkey.github.io/guide/creating-a-userscript/) - 创建用户脚本的完整指南，包含基础结构和最佳实践
- [Modern Syntax](https://violentmonkey.github.io/guide/using-modern-syntax/) - 现代语法指南，介绍 GM.* 异步 API 的使用方法
- [Observing DOM](https://violentmonkey.github.io/guide/observing-dom/) - DOM 观察技术，处理动态加载内容的必备指南
- [Keyboard Shortcuts](https://violentmonkey.github.io/guide/keyboard-shortcuts/) - 键盘快捷键配置与使用

### API 参考
- [Privileged APIs](https://violentmonkey.github.io/api/gm/) - 特权 API 完整参考，包含 GM_setValue、GM_xmlhttpRequest 等所有 API 的详细说明
- [Metadata Block](https://violentmonkey.github.io/api/metadata-block/) - 元数据块完整参考，包含所有 @指令 的详细说明
- [Matching](https://violentmonkey.github.io/api/matching/) - URL 匹配规则参考，@match 和 @include 的语法详解

## 社区资源

### GitHub 仓库
- [Violentmonkey 主仓库](https://github.com/violentmonkey/violentmonkey) - 源代码、问题反馈、功能请求
- [Violentmonkey 示例脚本](https://github.com/violentmonkey/violentmonkey/tree/master/examples) - 官方示例脚本集合

### 用户脚本托管平台
- [Greasy Fork](https://greasyfork.org/) - 最大的用户脚本托管平台，可搜索和发布脚本
- [OpenUserJS](https://openuserjs.org/) - 另一个流行的用户脚本托管平台

### 相关工具
- [Tampermonkey 文档](https://www.tampermonkey.net/documentation.php) - 另一个主流脚本管理器的文档，API 大部分兼容
- [Greasemonkey 维基](https://wiki.greasespot.net/) - Greasemonkey 官方维基，包含大量技术细节
