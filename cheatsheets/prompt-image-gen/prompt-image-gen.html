🍌 一眼入口 >>>

这不是模型介绍，而是一份给画面描述补词、查词、拼词的提示词手册。主用法有两种：从零拼装一段完整画面描述，或者拿已有 prompt 来补漏掉的视觉信息。

🧩 最小拼装顺序卡 >>>

从零写 prompt 时，按这个顺序补，通常比一口气乱堆词更稳。把它类比成搭页面：先有主结构，再补视觉层和细节层。

-> 构图

👤 主体词典

主体是在补“谁 / 什么是画面中心”。不要只写“一个女孩”“一个产品”，要补年龄感、状态、穿着、身份感或物体属性。
  • 人物类型： 年轻女性、少年感、成熟男性、儿童感、老人感、上班族、学生、研究员、机械师、厨师、模特、旅人、舞者、战士、流浪者
  • 外形与气质： 清冷感、松弛感、锋利感、脆弱感、疲惫感、精致感、野性气质、书卷气、未来感、街头感、贵气、纪实感
  • 脸部与发型： 短发、长卷发、湿发、凌乱发丝、高马尾、空气刘海、利落短发、轮廓分明、五官柔和、眼神锐利、目光空洞、轻微雀斑
  • 服装与配饰： 机能夹克、长风衣、皮夹克、廓形西装、丝质长裙、针织毛衣、工装裤、银色耳机、细框眼镜、金属项链、皮质手套
  • 非人物主体： 金属装置、机械臂、复古相机、玻璃瓶、香水瓶、耳机、手表、跑车、悬浮设备、未来终端、甜点、咖啡杯、建筑模型
  • 主体身份 + 外形气质 + 服装配饰：比只堆外貌更稳定

🖼️ 构图词典

构图是在补画面组织方式，决定主体放哪、画面是满是空、前中后景怎么排。
  • 主体位置： 居中构图、偏左构图、偏右构图、画面下方、画面上方、主体靠近边缘、主体位于黄金分割点
  • 画面结构： 对称构图、非对称构图、对角线构图、三角形构图、放射式构图、层层递进、框中框构图
  • 空间关系： 明确前景、中景、远景、背景压低存在感、前景遮挡、远处有轮廓、背景留白充足
  • 疏密与留白： 大面积留白、画面紧凑、信息密集、边缘干净、主体周围留出呼吸感、视觉重心集中
  • 画幅倾向： 竖构图、横构图、方形构图、超宽画幅、海报式版面、封面式构图
  • 主体位置 + 前中后景 + 留白程度：往往比单写“高级构图”更有用

🎥 镜头词典

镜头是在补“观众从哪看过去”。它是普通描述和电影感描述之间最明显的分水岭。
  • 景别： 超远景、远景、全身、中景、半身、近景、特写、极近特写、微距视角
  • 机位： 平视、俯拍、仰拍、低机位、高机位、贴近地面视角、肩后视角、侧后方视角
  • 焦距感： 广角感、长焦压缩感、轻微透视夸张、空间被压扁、背景被拉远、镜头更贴脸
  • 景深： 背景虚化、浅景深、深景深、焦点清晰、边缘渐虚、前景失焦、背景朦胧
  • 影像质感： 纪实摄影、电影镜头感、广告摄影、静物棚拍、抓拍感、监控视角、杂志大片感
  • 景别 + 机位 + 景深：三件套最容易直接提升镜头感

💡 光线词典

光线是在补“画面为什么成立”。很多平淡图不是词不够，而是没把主光方向、光质和空气感说清楚。
  • 主光方向： 顺光、逆光、侧光、顶光、底光、斜侧光、窗边侧光、背后轮廓光
  • 光质： 柔和漫射光、硬朗直射光、柔光箱质感、清晨柔光、阴天散射光、影棚均匀光
  • 对比关系： 高反差、低反差、暗部压深、亮部柔和、局部高光明显、整体明暗克制
  • 边缘与体积： 边缘光、轮廓光、体积光、穿透雾气的光束、光线切过灰尘、空气里有漂浮光尘
  • 环境与特效光： 霓虹光、荧光灯冷光、钨丝灯暖光、街灯昏黄光、屏幕冷光、火光反射、水面反光
  • 高频短语： 柔和光晕、细微高光、潮湿反光、冰冷反射、金属反射发冷、玻璃边缘泛光、逆光发丝、皮肤有轻微油润反光
  • 逆光 + 环境雾气 + 细小颗粒：很容易出电影感

🎨 色彩词典

色彩是在补“情绪温度”。不要只写红蓝紫，最好写冷暖倾向、饱和度和色彩关系。
  • 冷暖倾向： 偏冷色调、偏暖色调、冷暖对比、整体发冷、整体发暖、局部暖光点缀
  • 饱和度： 低饱和、轻度褪色、高饱和、克制饱和度、灰雾感、色彩压低
  • 常用色感： 冰蓝、雾灰蓝、霓虹紫、暖橙、奶油白、酒红、深墨绿、浅金色、冷青色、钨丝灯黄
  • 情绪色彩： 冷蓝孤独、暖橙舒适、灰绿色压抑、深红危险、金色希望、蓝紫神秘、黑金奢华
  • 搭配关系： 单色系、邻近色、互补色、局部撞色、背景压暗、主体颜色跳出画面
  • 色彩倾向 + 饱和度 + 一组主色：通常就够，不要一次报十种颜色

✨ 材质与细节词典

这是最容易把“普通图”拉成“可信图”的维度。重点不是写材质名本身，而是补触感、反光和表面痕迹。
  • 金属： 拉丝金属、抛光金属、氧化金属、冰冷金属表面、金属边缘磨损、螺丝细节、接缝清晰
  • 玻璃： 磨砂玻璃、透明玻璃、带冷凝水珠的玻璃、反光玻璃、边缘折射、模糊透视
  • 布料与皮革： 粗织物纹理、柔软棉布、丝绸反光、天鹅绒质感、旧皮革、折痕明显、布料被风轻微吹起
  • 木头与石材： 风化木纹、深色胡桃木、潮湿木面、粗糙石面、抛光石材、边缘磨圆、微微起灰
  • 液体与空气： 水珠附着、冷凝水、液体飞溅、蒸汽上升、环境雾气、薄薄烟雾、潮湿空气
  • 表面痕迹： 尘埃颗粒、胶片颗粒、微小划痕、指纹痕迹、轻微磨损、边缘掉漆、油污残留、细小裂纹、微量污渍、使用痕迹明显
  • 微观质感： 细微高光、柔和反射、局部高光跳点、表面不均匀反光、细密纹理、轻微颗粒感、材质层次清晰
  • 材质名 + 反光方式 + 表面痕迹：最能补真实度

🌫️ 氛围词典

氛围是在补空气感和情绪线。它不是风格名，而是“看上去像什么情绪正在发生”。
  • 冷清： 寂静、空旷、无人感、清晨冷意、寒冷空气、距离感强
  • 温暖： 松弛、舒缓、家居温度、灯光温柔、安静治愈、微热空气
  • 压抑： 闷、低气压、紧绷、沉重、缺氧感、灰暗、危险逼近
  • 神秘： 半遮半掩、难以看清、雾中轮廓、光线暧昧、未知感
  • 梦幻： 柔焦、朦胧、漂浮感、轻盈、幻觉感、发光颗粒
  • 纪实： 不修饰、现场感、真实生活痕迹、自然脏感、非摆拍
  • 末世 / 废土： 破败、风沙、尘土、荒凉、废弃感、幸存者氛围
  • 氛围词 + 色彩词 + 天气词：更容易形成稳定情绪

🏃 动作与状态词典

动作是在补主体的“活性”。很多画面像摆拍，是因为只写了对象，没有写它正在做什么。
  • 静态动作： 站立、坐着、靠墙、低头、回头、凝视镜头、双手自然下垂、安静等待、手扶桌面
  • 动态动作： 行走、转身、奔跑、伸手、俯身、抬头、拨开头发、拿起物件、风吹起衣摆
  • 情绪状态： 若有所思、疲惫、警惕、平静、冷漠、轻微微笑、紧张、松弛、神情恍惚
  • 物体状态： 悬浮、倾斜摆放、边缘发光、正在旋转、表面轻微震动、液体正在滴落、蒸汽缓慢上升
  • 动作 + 视线方向 + 衣物 / 发丝状态：能让主体更像正在发生

💃 姿态词典 >>>

姿态是在补身体朝向、重心、手脚关系和镜头里的线条。它和“动作”不同，动作更偏发生了什么，姿态更偏身体怎么摆才自然、显线条、适合写真。
  • 站姿母类： 45 度站姿、三分之四侧身、重心落在后腿、一脚前一脚后、双腿交叉站立、一腿微屈一腿伸直、S 曲线站姿、顶胯站姿
  • 身体朝向： 微侧身、半侧身朝向镜头、肩线轻微错开、骨盆与肩膀反向打开、身体朝向镜头脸部略偏、一肩靠近镜头、回身看向镜头
  • 手部任务： 单手轻扶下巴、指尖轻触脸侧、手指轻撩发丝、单手扶颈侧、单手叉腰、双手轻搭腰间、双手轻握衣角、单手插袋、手部轻搭栏杆
  • 腿部与重心： 重心落在单腿、后腿承重、前腿轻点地、前后错步站立、脚尖朝外轻开、一脚向前探出半步、缓步停顿的腿部状态、侧腿伸出拉长线条
  • 坐姿母类： 坐在椅边、坐姿前倾、交叉腿坐姿、抱膝坐姿、靠椅背放松坐姿、一腿自然向前延伸、身体稍微侧坐
  • 靠倚与支撑： 靠墙站立、靠栏杆站立、靠桌面支撑、身体轻微后仰、单肩贴墙、门框边倚靠、窗边斜倚
  • 回望与转身： 过肩回望、身体转开眼神看回镜头、转身一瞬停顿、行走中回眸、回头看向远处
  • 姿态母类 + 重心细节 + 手部任务 + 视线方向：比只写“优雅姿势”稳定得多

🏙️ 场景词典

场景是在补可见环境，不要只写“背景很酷”。要把空间、天气、地面、远处元素写出来。
  • 城市空间： 雨夜街头、地铁车厢、天桥、狭窄巷道、霓虹商店门口、空旷停车场、写字楼大堂、屋顶边缘
  • 室内空间： 极简客厅、厨房台面、酒店房间、展厅、工作室、摄影棚、旧仓库、实验室
  • 自然空间： 森林边缘、雾中山路、海边礁石、草地、雪地、峡谷、湖畔、沙漠公路
  • 天气与环境： 小雨、暴雨、薄雾、晨雾、雪粒、强风、潮湿空气、热浪扭曲、夜色潮湿、阳光穿透云层
  • 地面与背景细节： 湿地反射、水坑、碎纸、玻璃反光、地面落叶、背景招牌虚化、远处车灯散开、空气里有漂浮尘粒
  • 地点 + 天气 + 地面状态 + 远处背景：比单写“街头”更完整

🧷 风格词典

风格是最后收口用的，不要一开始就堆太多。最好只用 1 到 2 个强风格锚点。
  • 写实导向： 写实摄影、纪实感、商业广告感、棚拍质感、电影剧照感
  • 复古导向： 胶片感、怀旧色偏、老照片质感、轻微褪色、复古杂志感
  • 未来导向： 未来工业感、赛博质感、科技海报感、界面化视觉、极简未来感
  • 艺术导向： 梦境感、舞台化、超现实、画意光影、静物艺术摄影
  • 1-2：一次只保留 个风格锚点，避免彼此打架

🔠 文字排版词典 >>>

做海报、封面和 banner 时，文字描述不能只写“加标题”，最好同时说明位置、层级和可读性。
  • 位置： 标题居中、标题靠上、标题靠左、右下角标签、小字放在底部、文字围绕主体留白排布
  • 层级： 主标题最大、副标题较小、角标更轻、说明文字克制、信息层级清晰
  • 气质： 无衬线、字重大、细字重、未来感、商业感、极简排版、杂志封面感、科技感
  • 可读性要求： 文字清晰可读、不要拥挤、不要弯曲变形、不要被主体遮挡、留白充足
  • 文案内容 + 位置 + 层级 + 可读性要求：要一起写

⛔ 负面限制词典

负面限制不是为了把 prompt 写得更长，而是为了收口。优先限制最容易出错的点。
  • 画面质量： 不要过曝、不要死黑、不要过度锐化、不要脏污噪点、不要廉价滤镜感
  • 主体问题： 不要多余人物、不要变形肢体、不要奇怪手指、不要脸部结构漂移、不要服装乱改
  • 构图问题： 不要背景过满、不要主体太小、不要画面拥挤、不要裁切掉关键部位
  • 风格问题： 不要卡通感、不要塑料质感、不要过度夸张、不要廉价科幻感
  • 文字问题： 不要错字、不要模糊文字、不要挤在一起、不要覆盖主体脸部
  • 2-5：每次优先写 个最关键限制，不要把负面列表拉得过长

📸 专题词库：人像

人像最容易缺的不是“美女”“帅哥”这种笼统词，而是脸部状态、皮肤质感、妆面、视线和环境关系。
  • 面部： 轮廓清晰、下颌线利落、眼下轻微疲惫、唇部湿润、轻微雀斑、自然肤感、皮肤有真实毛孔
  • 发型： 发丝被风吹散、湿发贴脸、碎发垂落、发尾微卷、头发有冷色反光
  • 妆面： 干净底妆、微光泽肌肤、眼妆克制、唇色偏裸、带一点冷感高光
  • 姿态： 侧身回头、目光越过镜头、肩膀微收、半身静止、手扶领口
  • 环境人像： 街头环境人像、窗边自然光人像、雨夜霓虹人像、地铁车厢人像、室内低照度人像

👠 专题词库：女性写真姿态 >>>

这张卡片专门补女性写真摄影里高频、稳定、适合直接拼进 prompt 的姿态短语，重点覆盖半身、全身、近景和环境人像。
  • 半身写真： 肩膀略微错开、躯干轻转 10 到 30 度、单手轻碰头发、单手扶包带、手指轻触脸侧、半身靠墙、轻微重心偏移、目光看向镜头外
  • 全身写真： 一脚承重一脚放松、一脚向前探出半步、缓步向前、行走中回眸、靠门框站立、身体形成斜线、裙摆或外套有轻微运动感、竖构图全身站姿
  • 特写 / 近景： 头部轻微侧转、轻微低头、微抬下巴、眼睑放松、嘴角轻微动作、手靠近脸但不过度遮挡、面部周围留出干净线条
  • 环境人像： 站在窗边看向留白方向、坐在咖啡馆角落轻扶杯子、扶栏杆看向远处、站在门洞里形成框景、人物偏三分线站位、姿态和环境方向保持一致
  • 视线与情绪： 直视镜头、低眸、回眸、看向侧前方、看向窗外、眼神略微游离、冷静表情、轻微浅笑、若有所思
  • 适合直接拼接的短语： 三分之二侧身、单手轻扶下巴、重心落在后腿、指尖轻撩发丝、回身看向镜头、双腿交叉站立、半侧身朝向镜头、行走中回眸、身体朝向留白方向
  • 半身写真 + 肩膀略微错开 + 单手轻碰头发 + 目光看向镜头外 + 柔和侧光 全身写真 + 重心落在后腿 + 一脚向前探出半步 + 行走中回眸 + 竖构图 近景特写 + 头部轻微侧转 + 手指轻触脸侧 + 低眸 + 细微高光 环境人像 + 站在窗边 + 身体朝向留白方向 + 看向窗外 + 自然光

🧴 专题词库：产品静物

产品图核心是材质、台面、反射、边缘光和商业陈列感。
  • 陈列方式： 居中展示、悬浮展示、台面摆放、阶梯式摆放、单品特写、群组排布
  • 商业质感： 干净高级、奢华感、科技感、极简感、品牌广告感、杂志硬照感
  • 台面与背景： 亚克力台面、石材底座、镜面反射台面、纯色无缝背景、渐变背景、柔焦背景
  • 材质补词： 瓶身有细微高光、玻璃边缘折射、金属边缘冷反光、表面没有脏污、包装压纹清晰

🎬 专题词库：电影感场景

电影感往往来自镜头、天气、空气颗粒和低照度环境的组合，而不是单独一句“电影感”。
  • 高频元素： 雨夜街头、远处车灯、潮湿地面、路面反射、霓虹边缘光、空气漂浮颗粒、薄雾、背后体积光
  • 影像质感： 低照度、轻微胶片颗粒、压暗阴影、局部高光跳点、颜色克制、长焦压缩感
  • 动作与叙事： 一个人走在街头、停在路灯下、回头看向远处、衣摆被风吹起、像刚经历一段事件

🪧 专题词库：海报与标题字 >>>

海报类最容易失败在“文字有了，但版式没有”。要把视觉中心和标题层级一起写。
  • 版式： 海报式纵向版面、主体占中、标题压上方、留出底部说明区、四周留白、视觉中心明确
  • 标题要求： 主标题醒目、字重大、短词标题、副标题更轻、角标克制、品牌名清晰
  • 视觉控制： 文字不遮挡主体脸部、文字边缘清晰、标题与主体形成呼应、整体不拥挤

🍰 专题词库：食物 / 饮品

食物画面重点是新鲜感、热气、水润感、近景质感和可食欲细节。
  • 质感： 表面油亮、酱汁缓慢流动、切面湿润、新鲜水珠、蒸汽上升、微微冒热气、奶油纹理清晰
  • 构图： 微距近景、局部特写、桌面摆盘、手持展示、背景虚化、食物占据视觉中心
  • 氛围： 温暖灯光、清晨餐桌感、咖啡馆气氛、干净食物摄影感

🛋️ 专题词库：室内空间

室内空间最怕只有“漂亮房间”四个字。要补材质、采光、尺度和陈设关系。
  • 空间类型： 极简客厅、酒店房间、现代厨房、展厅、书房、工作室、咖啡馆角落
  • 材质组合： 原木、亚麻布、石材、磨砂金属、玻璃隔断、暖色灯带、粗纹理墙面
  • 光线： 自然采光、窗边柔光、顶部氛围灯、角落辅助灯、傍晚室内暖光
  • 空间感： 线条干净、尺度开阔、层次分明、家具摆放克制、空气安静

🤖 专题词库：赛博 / 科幻

赛博题材不是只有霓虹，还需要冷金属、界面投影、磨损痕迹和空气层次。
  • 视觉元素： 冷色霓虹、界面投影、悬浮光幕、金属拼接、发光线条、工业结构、电子标识
  • 材质细节： 冰冷反射、金属磨损、边缘掉漆、表面细小划痕、潮湿光泽、雾中灯光
  • 空气与叙事： 潮湿夜色、轻微烟雾、远处广告牌虚焦、空间压迫感、未来都市噪声感

🌄 专题词库：自然风光 / 天气

自然场景最容易只写地点，不写天气和空气。真正拉开差距的是风、雾、云层和光线穿透方式。
  • 天空与云层： 厚重云层、破开的云缝光、暮色天边、暴雨前天空、晨光薄云、晚霞层次
  • 天气与空气： 晨雾、细雨、暴雨、雪粒、潮湿空气、冷风、海边湿雾、热浪轻微扭曲
  • 地表细节： 潮湿岩石、被雨打湿的草地、海浪白沫、泥土颗粒、山脊轮廓、湖面反光

🧠 常见缺词补全提示

如果你觉得 prompt “差一点”，通常不是差一个神奇咒语，而是漏掉了其中一个层级。
  • 场景
  • 构图
  • 镜头
  • 材质
  • 氛围
  • 视线方向
  • 位置 / 层级 / 可读性
  • 负面限制

⚠️ 最常见错误

这部分只保留最容易导致画面失控的错误，方便写 prompt 前快速自检。
  • 一句话塞太多细节，主次不分
  • 同时改太多变量，导致每轮都漂
  • 风格锚点过多，彼此冲突
  • 只写抽象词，不写可见元素
  • 只写“高级感”“电影感”，不写镜头和光线
  • 材质名写了，但没有反光、痕迹和触感
  • 背景只写“很酷”，没有地点、天气和地面状态
  • 负面限制过长，反而稀释重点
