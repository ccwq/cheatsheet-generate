🍌 一眼入口 >>>

这不是模型介绍，而是一份面向生图 prompt 的中文拼装词典。核心目标不是“堆形容词”，而是让你知道画面缺哪一层就补哪一层。
  • 适合两类用法：
    • 从零拼一个完整 prompt
    • 拿已有 prompt 查漏补缺
  • 主体 -> 场景 -> 构图 -> 镜头 -> 光线 -> 色彩 -> 材质 -> 氛围 -> 动作 -> 文字 -> 负面限制
  • 不知道先看哪：
    • 场景 / 前中后景 / 环境元素：先看
    • 构图 / 镜头 / 光线：先看
    • 材质与表面痕迹：先看
    • 色彩 / 氛围 / 天气：先看
    • 文字排版 / 标题控制：先看

🧭 怎么用这份词典

读法按“先搭骨架，再补视觉层，最后收口”来理解。可以把它类比成前端页面：先写结构，再补样式，最后修边界条件。
  • 主体 + 场景 + 构图
  • 镜头 + 光线 + 色彩 + 材质 + 氛围
  • 动作 / 姿态 + 文字 + 负面限制
  • 本页结构建议这样扫：
    • 最小拼装模板
    • 基础维度词典
    • 专题模板

🧩 最小拼装模板 >>>

从零写 prompt 时，先套模板再补词，通常比直接想到什么写什么更稳。
[主体]
in / at [场景]
[构图与主体位置]
[镜头 / 景别 / 机位 / 景深]
[光线]
[色彩]
[材质与细节]
[氛围]
[动作 / 姿态 / 视线]
[文字排版要求，可选]
[负面限制]
  • 极简版：
年轻女性，雨夜街头，半身偏左构图，侧光与霓虹反射，冷蓝低饱和，湿发与皮夹克有细微反光，安静疏离，回头看向镜头，不要多余人物，不要手部变形
  • 海报版：
未来感耳机产品，悬浮展示，居中海报式构图，冷色轮廓光，金属与玻璃折射清晰，深色背景留白，顶部大标题清晰可读，右下角小标签，不要文字变形，不要遮挡主体

🛠️ 画面出问题时先补哪一层

多数失败图不是“模型不行”，而是 prompt 在某一层描述严重缺失。先定位问题，再补最相关的层，不要一口气全改。
  • 场景 + 前中后景 + 地面 / 背景元素：补
  • 构图 + 景别 + 机位 + 景深：补
  • 主光方向 + 光质 + 阴影关系：补
  • 材质 + 反光方式 + 表面痕迹：补
  • 色彩倾向 + 饱和度 + 天气 / 空气：补
  • 姿态 + 重心 + 手部任务 + 视线：补
  • 位置 + 层级 + 留白 + 可读性：补

👤 基础维度词典：主体系统

主体系统回答“谁 / 什么是视觉中心”。这里优先补身份、状态和可见特征，而不是只堆抽象审美词。

人物主体

  • 年轻女性、少年感、成熟男性、儿童感、老人感、上班族、学生、研究员、机械师、厨师、模特、旅人、舞者、战士、流浪者

非人物主体

  • 金属装置、机械臂、复古相机、玻璃瓶、香水瓶、耳机、手表、跑车、悬浮设备、未来终端、甜点、咖啡杯、建筑模型

外形与气质

  • 清冷感、松弛感、锋利感、脆弱感、疲惫感、精致感、野性气质、书卷气、未来感、街头感、贵气、纪实感

脸部与发型

  • 短发、长卷发、湿发、凌乱发丝、高马尾、空气刘海、利落短发、轮廓分明、五官柔和、眼神锐利、目光空洞、轻微雀斑

服装与配饰

  • 机能夹克、长风衣、皮夹克、廓形西装、丝质长裙、针织毛衣、工装裤、银色耳机、细框眼镜、金属项链、皮质手套

组合提醒

  • 主体身份 + 外形气质 + 服装配饰
  • 如果主体总是漂： 先少写抽象气质，多写可见外形和穿着

💃 基础维度词典：姿态与动作 >>>

动作回答“正在做什么”，姿态回答“身体怎么摆”。两者一起写，人物更像正在发生，而不是摆拍站桩。

静态动作

  • 站立、坐着、靠墙、低头、回头、凝视镜头、双手自然下垂、安静等待、手扶桌面

动态动作

  • 行走、转身、奔跑、伸手、俯身、抬头、拨开头发、拿起物件、风吹起衣摆

情绪状态

  • 若有所思、疲惫、警惕、平静、冷漠、轻微微笑、紧张、松弛、神情恍惚

站姿母类

  • 45 度站姿、三分之四侧身、重心落在后腿、一脚前一脚后、双腿交叉站立、一腿微屈一腿伸直、S 曲线站姿、顶胯站姿

手部任务

  • 单手轻扶下巴、指尖轻触脸侧、手指轻撩发丝、单手扶颈侧、单手叉腰、双手轻搭腰间、双手轻握衣角、单手插袋、手部轻搭栏杆

腿部与重心

  • 重心落在单腿、后腿承重、前腿轻点地、前后错步站立、脚尖朝外轻开、一脚向前探出半步、缓步停顿的腿部状态、侧腿伸出拉长线条

坐姿与依靠

  • 坐在椅边、坐姿前倾、交叉腿坐姿、抱膝坐姿、靠椅背放松坐姿、一腿自然向前延伸、身体稍微侧坐、靠墙站立、靠栏杆站立、靠桌面支撑、单肩贴墙、窗边斜倚

组合提醒

  • 姿态母类 + 重心细节 + 手部任务 + 视线方向
  • 重心 手部任务：优先补 和

🏙️ 基础维度词典：场景与空间

场景不是“背景很酷”，而是把空间、天气、地面和远景元素说清楚，让主体真正落地。

城市空间

  • 雨夜街头、地铁车厢、天桥、狭窄巷道、霓虹商店门口、空旷停车场、写字楼大堂、屋顶边缘

室内空间

  • 极简客厅、厨房台面、酒店房间、展厅、工作室、摄影棚、旧仓库、实验室

自然空间

  • 森林边缘、雾中山路、海边礁石、草地、雪地、峡谷、湖畔、沙漠公路

天气与空气

  • 小雨、暴雨、薄雾、晨雾、雪粒、强风、潮湿空气、热浪扭曲、夜色潮湿、阳光穿透云层

地面与背景细节

  • 湿地反射、水坑、碎纸、玻璃反光、地面落叶、背景招牌虚化、远处车灯散开、空气里有漂浮尘粒

组合提醒

  • 地点 + 天气 + 地面状态 + 远处背景

🖼️ 基础维度词典：构图与画面组织

构图是在决定主体放哪、前后景怎么排、画面空还是满。它直接影响“像随手拍”还是“像被设计过”。

主体位置

  • 居中构图、偏左构图、偏右构图、画面下方、画面上方、主体靠近边缘、主体位于黄金分割点

画面结构

  • 对称构图、非对称构图、对角线构图、三角形构图、放射式构图、层层递进、框中框构图

前中后景

  • 明确前景、中景、远景、前景遮挡、远处有轮廓、背景压低存在感、主体从环境里跳出

疏密与留白

  • 大面积留白、画面紧凑、信息密集、边缘干净、主体周围留出呼吸感、视觉重心集中

画幅与版面

  • 竖构图、横构图、方形构图、超宽画幅、海报式版面、封面式构图、社媒封面比例、移动端竖图倾向

组合提醒

  • 主体位置 + 前中后景 + 留白程度

🎥 基础维度词典：镜头与视角

镜头回答“观众从哪里看过去”。这是普通描述和电影感描述之间最容易拉开差距的一层。

景别

  • 超远景、远景、全身、中景、半身、近景、特写、极近特写、微距视角

机位

  • 平视、俯拍、仰拍、低机位、高机位、贴近地面视角、肩后视角、侧后方视角

焦距感

  • 广角感、长焦压缩感、轻微透视夸张、空间被压扁、背景被拉远、镜头更贴脸

景深

  • 背景虚化、浅景深、深景深、焦点清晰、边缘渐虚、前景失焦、背景朦胧

影像质感

  • 纪实摄影、电影镜头感、广告摄影、静物棚拍、抓拍感、监控视角、杂志大片感

组合提醒

  • 景别 + 机位 + 景深

💡 基础维度词典：光线与色彩

光线决定画面是否成立，色彩决定情绪温度。很多“像样”和“有感觉”，其实都靠这两层一起完成。

主光方向

  • 顺光、逆光、侧光、顶光、底光、斜侧光、窗边侧光、背后轮廓光

光质与对比

  • 柔和漫射光、硬朗直射光、柔光箱质感、清晨柔光、阴天散射光、影棚均匀光、高反差、低反差、暗部压深、亮部柔和

环境光与体积感

  • 边缘光、轮廓光、体积光、穿透雾气的光束、光线切过灰尘、空气里有漂浮光尘、霓虹光、荧光灯冷光、钨丝灯暖光、屏幕冷光、火光反射、水面反光

冷暖倾向

  • 偏冷色调、偏暖色调、冷暖对比、整体发冷、整体发暖、局部暖光点缀

饱和度与色相

  • 低饱和、轻度褪色、高饱和、克制饱和度、灰雾感、色彩压低、冰蓝、雾灰蓝、霓虹紫、暖橙、奶油白、酒红、深墨绿、浅金色、冷青色

情绪色彩

  • 冷蓝孤独、暖橙舒适、灰绿色压抑、深红危险、金色希望、蓝紫神秘、黑金奢华

组合提醒

  • 光线方向 + 光质 + 色彩倾向 + 1 组主色
  • 逆光 + 环境雾气 + 低饱和冷色：试试

✨ 基础维度词典：材质、细节与氛围

这一层专门处理“真实感”和“空气感”。材质负责触感，氛围负责情绪，不要只写风格名而不写可见细节。

金属与玻璃

  • 拉丝金属、抛光金属、氧化金属、冰冷金属表面、金属边缘磨损、螺丝细节、接缝清晰、磨砂玻璃、透明玻璃、带冷凝水珠的玻璃、反光玻璃、边缘折射

布料与自然材质

  • 粗织物纹理、柔软棉布、丝绸反光、天鹅绒质感、旧皮革、折痕明显、风化木纹、深色胡桃木、潮湿木面、粗糙石面、抛光石材

液体与空气介质

  • 水珠附着、冷凝水、液体飞溅、蒸汽上升、环境雾气、薄薄烟雾、潮湿空气

表面痕迹

  • 尘埃颗粒、胶片颗粒、微小划痕、指纹痕迹、轻微磨损、边缘掉漆、油污残留、细小裂纹、微量污渍、使用痕迹明显

微观质感

  • 细微高光、柔和反射、局部高光跳点、表面不均匀反光、细密纹理、轻微颗粒感、材质层次清晰

氛围倾向

  • 寂静、空旷、无人感、清晨冷意、松弛、舒缓、家居温度、闷、低气压、紧绷、神秘、半遮半掩、梦幻、柔焦、纪实、不修饰、末世、废弃感

组合提醒

  • 材质名 + 反光方式 + 表面痕迹
  • 氛围词 + 色彩词 + 天气词

🔠 基础维度词典：风格、文字与收口 >>>

风格负责最后收口，文字和负面限制负责控制边界。这里要克制，不要把所有想法一次塞进来。

风格锚点

  • 写实摄影、纪实感、商业广告感、棚拍质感、电影剧照感、胶片感、怀旧色偏、复古杂志感、未来工业感、赛博质感、科技海报感、梦境感、超现实、静物艺术摄影

文字位置与层级

  • 标题居中、标题靠上、标题靠左、右下角标签、小字放在底部、文字围绕主体留白排布、主标题最大、副标题较小、角标更轻、说明文字克制

文字气质与可读性

  • 无衬线、字重大、细字重、未来感、商业感、极简排版、杂志封面感、科技感、文字清晰可读、不要拥挤、不要弯曲变形、不要被主体遮挡、留白充足

负面限制

  • 不要过曝、不要死黑、不要过度锐化、不要廉价滤镜感、不要多余人物、不要变形肢体、不要奇怪手指、不要脸部结构漂移、不要背景过满、不要主体太小、不要卡通感、不要塑料质感、不要错字、不要模糊文字

组合提醒

  • 1-2
  • 2-5

📸 高频专题模板：人像

人像类最容易缺的是脸部状态、皮肤质感、姿态和环境关系，不是“美女”“高级感”这种笼统标签。

高频补词

  • 面部： 轮廓清晰、下颌线利落、眼下轻微疲惫、唇部湿润、轻微雀斑、自然肤感、皮肤有真实毛孔
  • 发型： 发丝被风吹散、湿发贴脸、碎发垂落、发尾微卷、头发有冷色反光
  • 妆面： 干净底妆、微光泽肌肤、眼妆克制、唇色偏裸、带一点冷感高光
  • 环境： 街头环境人像、窗边自然光人像、雨夜霓虹人像、地铁车厢人像、室内低照度人像

直接拼接模板

年轻女性，街头环境人像，半身偏左构图，肩膀轻微错开，湿发贴脸，窗边侧光 / 雨夜霓虹反射，冷蓝低饱和，自然肤感与真实毛孔，安静疏离，目光越过镜头，不要多余人物，不要手部变形

👠 高频专题模板：女性写真姿态 >>>

这张卡片专门收拢可直接拼进 prompt 的女性写真姿态短语，适合半身、全身、近景和环境人像。

半身写真

  • 肩膀略微错开、躯干轻转 10 到 30 度、单手轻碰头发、单手扶包带、手指轻触脸侧、半身靠墙、轻微重心偏移、目光看向镜头外

全身写真

  • 一脚承重一脚放松、一脚向前探出半步、缓步向前、行走中回眸、靠门框站立、身体形成斜线、裙摆或外套有轻微运动感、竖构图全身站姿

特写 / 近景

  • 头部轻微侧转、轻微低头、微抬下巴、眼睑放松、嘴角轻微动作、手靠近脸但不过度遮挡、面部周围留出干净线条

环境人像

  • 站在窗边看向留白方向、坐在咖啡馆角落轻扶杯子、扶栏杆看向远处、站在门洞里形成框景、人物偏三分线站位、姿态和环境方向保持一致

适合直接拼接的短语

  • 三分之二侧身、单手轻扶下巴、重心落在后腿、指尖轻撩发丝、回身看向镜头、双腿交叉站立、半侧身朝向镜头、行走中回眸、身体朝向留白方向

🧴 高频专题模板：产品静物

产品图核心不是“高级”，而是台面、材质、反射、边缘光和品牌陈列逻辑。

高频补词

  • 陈列方式： 居中展示、悬浮展示、台面摆放、阶梯式摆放、单品特写、群组排布
  • 商业质感： 干净高级、奢华感、科技感、极简感、品牌广告感、杂志硬照感
  • 台面与背景： 亚克力台面、石材底座、镜面反射台面、纯色无缝背景、渐变背景、柔焦背景
  • 材质补词： 瓶身有细微高光、玻璃边缘折射、金属边缘冷反光、表面没有脏污、包装压纹清晰

直接拼接模板

香水瓶产品静物，居中展示，海报式纵向构图，镜面反射台面，背后轮廓光与柔和高光，冷白与浅金色搭配，玻璃边缘折射清晰，品牌广告感，顶部主标题清晰可读，不要文字变形，不要背景杂乱

🎬 高频专题模板：电影感场景

电影感通常来自镜头、天气、空气颗粒和低照度环境的组合，不是一句“电影感”本身。

高频元素

  • 雨夜街头、远处车灯、潮湿地面、路面反射、霓虹边缘光、空气漂浮颗粒、薄雾、背后体积光

影像质感

  • 低照度、轻微胶片颗粒、压暗阴影、局部高光跳点、颜色克制、长焦压缩感

动作与叙事

  • 一个人走在街头、停在路灯下、回头看向远处、衣摆被风吹起、像刚经历一段事件

🪧 高频专题模板：海报与标题字 >>>

海报类最常见的失败点不是“没有字”，而是没有把视觉中心、标题层级和留白关系一起交代。

版式控制

  • 海报式纵向版面、主体占中、标题压上方、留出底部说明区、四周留白、视觉中心明确

标题要求

  • 主标题醒目、字重大、短词标题、副标题更轻、角标克制、品牌名清晰

视觉控制

  • 文字不遮挡主体脸部、文字边缘清晰、标题与主体形成呼应、整体不拥挤

直接拼接模板

科技海报，主体居中，顶部大标题，底部小号说明文字，右下角标签，深色背景留白，文字清晰可读，标题不要弯曲变形，不要覆盖主体，不要排版拥挤

🍰 高频专题模板：食物 / 饮品

食物类重点是新鲜感、热气、水润感和近景可食欲细节，别只写“好吃”“精致”。

高频补词

  • 质感： 表面油亮、酱汁缓慢流动、切面湿润、新鲜水珠、蒸汽上升、微微冒热气、奶油纹理清晰
  • 构图： 微距近景、局部特写、桌面摆盘、手持展示、背景虚化、食物占据视觉中心
  • 氛围： 温暖灯光、清晨餐桌感、咖啡馆气氛、干净食物摄影感

🛋️ 高频专题模板：室内空间

室内空间最怕只有“漂亮房间”四个字。要补材质、采光、尺度和陈设关系。

空间类型

  • 极简客厅、酒店房间、现代厨房、展厅、书房、工作室、咖啡馆角落

材质组合

  • 原木、亚麻布、石材、磨砂金属、玻璃隔断、暖色灯带、粗纹理墙面

光线与空间感

  • 自然采光、窗边柔光、顶部氛围灯、角落辅助灯、傍晚室内暖光、线条干净、尺度开阔、层次分明、家具摆放克制、空气安静

🤖 高频专题模板：赛博 / 科幻

赛博题材不只是霓虹，还要补冷金属、界面投影、磨损痕迹和空气层次。

视觉元素

  • 冷色霓虹、界面投影、悬浮光幕、金属拼接、发光线条、工业结构、电子标识

材质细节

  • 冰冷反射、金属磨损、边缘掉漆、表面细小划痕、潮湿光泽、雾中灯光

空气与叙事

  • 潮湿夜色、轻微烟雾、远处广告牌虚焦、空间压迫感、未来都市噪声感

🌄 高频专题模板：自然风光 / 天气

自然场景最容易只写地点，不写天气和空气。真正拉开差距的是风、雾、云层和光线穿透方式。

天空与云层

  • 厚重云层、破开的云缝光、暮色天边、暴雨前天空、晨光薄云、晚霞层次

天气与空气

  • 晨雾、细雨、暴雨、雪粒、潮湿空气、冷风、海边湿雾、热浪轻微扭曲

地表细节

  • 潮湿岩石、被雨打湿的草地、海浪白沫、泥土颗粒、山脊轮廓、湖面反光

🧱 高频组合模板

当你不想从零拼时，直接抄这类“骨架稳定”的组合模板，再替换主体和场景即可。
  • 主体身份 + 半身 / 全身 + 侧光 + 低饱和冷暖关系 + 真实皮肤 / 发丝细节 + 目光方向
  • 单品主体 + 居中 / 悬浮展示 + 轮廓光 + 材质折射 / 反光 + 干净背景 + 标题层级
  • 人物 + 城市场景 + 长焦 / 低照度 + 雨雾颗粒 + 压暗阴影 + 一个动作
  • 食物主体 + 微距 / 特写 + 温暖灯光 + 水润 / 热气 + 干净背景虚化
  • 空间类型 + 材质组合 + 自然采光 / 氛围灯 + 干净线条 + 留白

⚠️ 最常见错误

这部分只保留最容易导致画面失控的错误，方便写 prompt 前快速自检。
  • 一句话塞太多细节，主次不分
  • 同时改太多变量，导致每轮都漂
  • 风格锚点过多，彼此冲突
  • 只写抽象词，不写可见元素
  • 只写“高级感”“电影感”，不写镜头和光线
  • 材质名写了，但没有反光、痕迹和触感
  • 背景只写“很酷”，没有地点、天气和地面状态
  • 负面限制过长，反而稀释重点
