🔧 常用配置和技巧 >>>

提升效率的技巧。

Shell wrapper

function y() {
    local tmp="$(mktemp -t "yazi-cwd.XXXXXX")"
    cwd
    command yazi "$@" --cwd-file="$tmp"
    IFS= read -r -d '' cwd < "$tmp"
    [ "$cwd" != "$PWD" ] && [ -d "$cwd" ] && builtin cd -- "$cwd"
    rm -f -- "$tmp"
}

快速跳转

  • z：通过 zoxide 跳转
  • Z：通过 fzf 跳转
  • g Space：交互式选择目录
