一眼入口

x-cmd 是一个统一的命令行工具调用层，通过 x <module> [子命令] 访问 370+ 预置工具模块。
# 安装
npm install -g x-cmd

# 基本用法
x <模块> [子命令] [参数]

# 查看帮助
x help <模块>

# 离线文档
x --help
x <模块> -h

通用选项

选项说明
-h, --help显示帮助
-v, --version显示版本
--dry-run预览执行（不实际运行）
-q, --quiet静默输出
-d 变量=值1,值2批量展开
--concurrency N并发数
--timeout N超时（秒）
--verbose详细输出
--jsonJSON 输出
--table表格输出
