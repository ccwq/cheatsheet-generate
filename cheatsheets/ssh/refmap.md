# ssh 参考资源

## 官方文档

- [OpenSSH Portable](https://www.openssh.com/portable.html) - OpenSSH Portable 官方入口与版本发布说明。
- [ssh(1)](https://man.openbsd.org/ssh) - `ssh` 客户端主手册，覆盖登录、执行命令、转发、跳板等。
- [ssh_config(5)](https://man.openbsd.org/ssh_config.5) - 客户端配置文件说明，覆盖 `Host`、`ProxyJump`、`ControlMaster` 等。
- [ssh-agent(1)](https://man.openbsd.org/ssh-agent.1) - SSH agent 说明。
- [ssh-add(1)](https://man.openbsd.org/ssh-add.1) - 向 agent 添加/移除密钥。
- [ssh-keygen(1)](https://man.openbsd.org/ssh-keygen.1) - 密钥生成、指纹、`known_hosts` 维护。
- [ssh-keyscan(1)](https://man.openbsd.org/ssh-keyscan.1) - 批量采集远端主机公钥指纹。
- [scp(1)](https://man.openbsd.org/scp.1) - 安全复制命令。
- [sftp(1)](https://man.openbsd.org/sftp.1) - SFTP 客户端。

## 相关主题

- [OpenSSH FAQ](https://www.openssh.com/faq.html) - 常见问题与使用说明。
- [OpenSSH 安全说明](https://www.openssh.com/security.html) - 安全公告入口。
- [OpenSSH 发布日志](https://www.openssh.com/releasenotes.html) - 版本更新记录。
