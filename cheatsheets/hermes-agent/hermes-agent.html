安全默认值

主题建议
不可信代码用 docker / ssh 后端
危险命令保持 approvals.mode: manual 或 smart
消息网关用 allowlist 或 DM pairing，不要开全量放行
密钥放 ~/.hermes/.env，不要写进仓库
容器前向变量只转发你能接受暴露的变量
密钥泄露security.redact_secrets: true 自动脱敏
MCP 安全MCP OAuth 2.1 PKCE + OSV 恶意软件扫描
浏览器泄露URL 与 LLM 回复扫描密钥模式，阻止泄露
