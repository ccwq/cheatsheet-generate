# Git版本控制系统

## 配置
- config: 全局或局部配置
- init: 初始化仓库
- clone: 克隆仓库

## 文件操作
- add: 暂存文件
- status: 查看状态
- diff: 查看差异
- rm: 删除文件
- mv: 移动文件

## 提交
- commit: 提交更改
- amend: 修改提交
- log: 查看历史
- show: 显示提交详情

## 分支
- branch: 分支管理
- checkout: 切换分支
- switch: 分支切换(Git 2.23+)
- merge: 合并分支

## 远程
- remote: 远程仓库管理
- fetch: 获取远程更新
- pull: 拉取并合并
- push: 推送到远程

## 标签
- tag: 创建和管理标签

## 暂存
- stash: 临时保存工作

## 重置
- reset: 重置状态
- restore: 恢复文件(Git 2.23+)
- revert: 创建反向提交

## 子模块
- submodule: 管理子模块

## 高级
- rebase: 变基操作
- cherry-pick: 选择提交
- bisect: 二分查找
- blame: 追踪修改
- grep: 内容搜索

## 日常工作流
- 创建分支并切换: checkout -b
- 快速提交: commit -am
- 查看日志: log --oneline
- 强制推送: push --force-with-lease
