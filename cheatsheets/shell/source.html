Bash 与 Shell 常用命令与脚本速查表

文件与目录操作

核心命令

  • ls -alh 列出所有文件，包括隐藏文件，使用人类可读大小
  • cd /path/to/dir 切换当前工作目录
  • pwd 显示当前工作目录完整路径
  • mkdir -p /path/to/dir 递归创建目录
  • rm -rf /path 递归强制删除目录及文件，谨慎使用
  • cp -r src/ dest/ 复制目录及其内容
  • mv oldname newname 重命名或移动文件/目录
  • touch file.txt 创建空文件或更新文件时间戳

文件查看与编辑

核心命令

  • cat file.txt 输出文件内容
  • less file.txt 分页查看文件，可上下翻页
  • head -n 20 file.txt 查看文件前20行
  • tail -f /var/log/syslog 实时跟踪文件追加
  • vi file.txt 使用vi编辑文件
  • nano file.txt 使用nano编辑文件

查找与定位

核心命令

  • find /path -name '*.log' 在指定目录查找文件
  • locate filename 快速查找文件（依赖 updatedb）
  • which command 显示命令绝对路径
  • whereis command 显示源代码和手册路径

管道与重定向

核心命令

  • command1 | command2 管道，将 STDOUT 传递给下一个命令
  • command > out.txt 将 STDOUT 重定向到文件，覆盖
  • command >> out.txt 将 STDOUT 追加到文件
  • command 2> err.txt 将 STDERR 重定向到文件
  • command &> all.txt 将 STDOUT 和 STDERR 重定向到同一文件
  • command < in.txt 从文件读取 STDIN
  • tee file.txt 将输出同时写入文件和 STDOUT

文本处理

核心命令

  • grep 'pattern' file.txt 模式搜索
  • sed 's/old/new/g' file.txt 文本替换
  • awk '{print $2}' file.txt 文本字段处理
  • cut -d':' -f1-3 /etc/passwd 按定界符分割并提取字段
  • sort file.txt 排序文件内容
  • uniq -c file.txt 去重并计数
  • tr '[:upper:]' '[:lower:]' 大小写转换

进程管理

核心命令

  • ps aux 列出所有进程
  • top 实时查看进程状态
  • htop 交互式进程查看（需安装）
  • kill -SIGTERM PID 发送 TERM 信号终止进程
  • kill -9 PID 强制杀死进程
  • bg 将暂停的作业放到后台
  • fg 将后台作业切回前台

用户与权限

核心命令

  • sudo command 以 root 权限运行命令
  • su - user 切换用户并加载环境
  • chmod 755 script.sh 修改文件权限
  • chown user:group file 修改文件所有者
  • id user 查看用户及其组信息
  • groups user 查看用户所属组

网络操作

核心命令

  • ping host 测试主机连通性
  • curl -I https://example.com 查看 HTTP 响应头
  • wget https://example.com/file 下载文件
  • ssh user@host 远程登录
  • scp file user@host:/path 复制文件到远程
  • netstat -tuln 查看端口占用
  • ip addr show 显示网络接口和地址

环境变量与配置

核心命令

  • export VAR=value 设置环境变量
  • echo $VAR 显示变量值
  • env 列出所有环境变量
  • source ~/.bashrc 重新加载配置文件
  • alias ll='ls -alh' 定义别名
  • unalias ll 移除别名

历史与作业控制

核心命令

  • history 查看命令历史
  • !n 重复执行历史编号为 n 的命令
  • !! 重复执行上一条命令
  • Ctrl + R 反向搜索历史命令
  • jobs 列出后台作业
  • kill %1 杀死作业号为 1 的作业

Shell 快捷键

常用快捷键

  • Ctrl + A 光标移动到行首
  • Ctrl + E 光标移动到行尾
  • Ctrl + U 删除光标前所有字符
  • Ctrl + K 删除光标后所有字符
  • Ctrl + L 清屏 (等同于 clear)
  • Ctrl + C 终止当前命令
  • Ctrl + Z 暂停当前命令并置于后台
  • Tab 命令或路径自动补全

Shell 脚本基础

核心要素

  • #!/usr/bin/env bash Shebang 声明
  • VAR=value 变量定义；使用 "${VAR}" 引用
  • # 单行注释 注释格式
  • if…fi、case…esac、for…do…done、while…do…done、until…do…done 流程控制
  • func_name() { … } 定义函数；用 func_name 调用
  • exit N 与 $? 退出状态与返回值

参数扩展与替换

常用用法

  • ${parameter} 基本扩展
  • ${parameter:-word} 参数为空时使用默认值
  • ${parameter:=word} 参数为空时赋值并返回
  • ${parameter:?word} 参数为空时报错并退出
  • ${parameter:+word} 参数已设置时使用替换值
  • ${parameter:offset} 从指定偏移截取子串
  • ${parameter:offset:length} 截取指定长度子串
  • ${parameter#pattern} 从前删除最短匹配
  • ${parameter##pattern} 从前删除最长匹配
  • ${parameter%pattern} 从后删除最短匹配
  • ${parameter%%pattern} 从后删除最长匹配
  • ${parameter/pattern/string} 第一次模式替换
  • ${parameter//pattern/string} 全局模式替换
  • ${parameter^pattern} 首字母大写
  • ${parameter^^pattern} 全部大写
  • ${parameter,pattern} 首字母小写
  • ${parameter,,pattern} 全部小写

通配符与 Brace Expansion

常用用法

  • * 匹配零个或多个字符
  • ? 匹配单个字符
  • [chars] 匹配方括号内任一字符
  • [!chars] 匹配不在方括号内任一字符
  • {item1,item2} 列举扩展
  • {n..m} 范围序列
  • {n..m..step} 指定步长序列

Here 文档与命令替换

使用方法

  • <<EOF … EOF 标准 Here 文档
  • <<-'EOF' 去除缩进的 Here 文档
  • <<"EOF" 禁用扩展的 Here 文档
  • $(command) 命令替换
  • `command` 旧式命令替换

压缩与归档

常用工具与参数

  • tar -cvf archive.tar 文件/目录 创建 tar 归档
  • tar -xvf archive.tar 解压 tar 归档
  • tar -czvf archive.tar.gz 文件/目录 创建 gzip 归档
  • tar -xjvf archive.tar.bz2 解压 bzip2 归档
  • gzip file.txt 压缩为 .gz 文件
  • gunzip file.txt.gz 解压 .gz 文件
  • zip archive.zip 文件/目录 创建 zip 压缩包
  • unzip archive.zip 解压 zip 压缩包

提示符与脚本健壮性

配置项

  • PS1="\\u@\\h \\w$ " 自定义提示符，显示用户、主机和路径
  • set -e 脚本遇错即退出
  • set -u 未定义变量时报错退出
  • set -o pipefail 管道任一命令失败时返回非零状态
  • trap '命令' ERR EXIT 捕获错误和脚本退出

调试与跟踪

调试选项

  • bash -x script.sh 以跟踪模式执行脚本
  • bash -v script.sh 打印脚本命令行到 STDOUT
  • bash -n script.sh 语法检查
  • export PS4='+ $LINENO: ' 设置调试前缀，显示行号
